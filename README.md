Консольная программа на C++ для расчета параметров прямоугольника, трапеции, окружности, треугольника.

Функциональность

Вычисление периметра прямоугольника  
Вычисление площади прямоугольника  
Вычисление длины диагонали (по теореме Пифагора)  
Вычисление периметра трапеции  
Вычисление площади трапеции  
Вычисление средней линии трапеции  
Вычисление длинны окружности  
Вычисление площади окружности  
Вычисление площади сектора окружности  
Вычисление периметра треугольника 
Вычисление площади треугольника по формуле Герона 
Проверка треугольника на равнобедренность  \
